# 📝 Python Modern To-Do Uygulaması

Python ve `tkinter` kütüphanesi kullanılarak geliştirilmiş, şık tasarımlı ve kullanımı kolay bir "Yapılacaklar Listesi" (To-Do List) masaüstü uygulaması. 

Ekstra hiçbir kütüphane kurulumu gerektirmez, Python'un yüklü olduğu her bilgisayarda anında çalışır!

## ✨ Özellikler

* **Modern Arayüz:** Göz yormayan koyu tema (dark mode) ve özel renk paleti.
* **Hızlı Kullanım:** Görev ekleme, seçili görevi silme ve tüm listeyi tek tuşla temizleme.
* **Kullanıcı Dostu Uyarılar:** Boş görev eklemeye veya seçim yapmadan silmeye çalıştığınızda uygulama çökmez, sizi bilgilendirir.
* **Kaydırma Çubuğu:** Liste uzadığında görevler arasında rahatça gezinebilmeniz için entegre scrollbar.
